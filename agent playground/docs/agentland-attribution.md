# AgentLand attribution

AgentLand began from the open-source Generative Agents research environment by
Joon Sung Park and collaborators. The inherited repository remains under its
Apache-2.0 licence and retains all upstream notices.

AgentLand P0 adds an isolated Django application for accountable productive
worker events, independent file evidence, bounded work execution, and durable
task/session audit state. It does not publicly reuse the inherited Smallville
maps, character sprites, personas, fictional biographies, or prompt library.
